# ToDo Immersive

Todo list created with vanilla JavaScript

### Prerequisites

```
nodejs
```

### Installing

To install the dependencies:

```
npm install
```

To run the project:

```
npm start
```

* [Html] - Creating the web page
* [Css] - Styling
